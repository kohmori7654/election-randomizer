# Phase 25: グリッド座標修正 - 四国・沖縄の重なり解消

## 背景・問題

P21 で `offsetX` を導入した結果、以下の2箇所で新たな重なりが発生。

| 競合ペア | X 重複 | Y 重複 | 原因 |
|---|---|---|---|
| 沖縄(row11,col0) ↔ 鹿児島(row10-11,col1,offsetX:-20) | 18px | 44px | 九州に offsetX:-20 を付与したが沖縄は据え置き |
| 香川(row10,col4,offsetX:+20) ↔ 兵庫(row9-10,col5,h:2) | 18px | 44px | offsetX:+20 で香川が右シフトし兵庫に衝突 |

ユーザー要件（2026-06-16）:
> 沖縄が九州と重なっている、沖縄を九州から左下に離して
> 四国が本州と繋がっており、兵庫と重なっている、四国を左下に離して

## 修正方針

### 四国修正（香川 ↔ 兵庫 の重なり解消）

`offsetX` を廃止し、代わりに **row を1段下げる** ことで瀬戸内海の空白を作る。

| 都道府県 | 現在 | 修正後 |
|---|---|---|
| 愛媛 | row 10, col 3, offsetX: 20 | row 11, col 3（offsetX 削除） |
| 香川 | row 10, col 4, offsetX: 20 | row 11, col 4（offsetX 削除） |
| 高知 | row 11, col 3, offsetX: 20 | row 12, col 3（offsetX 削除） |
| 徳島 | row 11, col 4, offsetX: 20 | row 12, col 4（offsetX 削除） |

**効果試算**（計算済み）:
- 中国地方 広島(row9) 下端 y=458 → 愛媛(row11) 上端 y=506 → **48px 空白（瀬戸内海）**
- 香川(row11) 下端 y=550 vs 兵庫(row9-10) 下端 y=504 → **Y方向に重なりなし**
- 香川 右端 x=248 vs 兵庫 左端 x=250 → **X方向も 2px 空き（衝突解消）**

### 沖縄修正（沖縄 ↔ 鹿児島 の重なり解消）

row を 1〜2 段下げる。

| 都道府県 | 現在 | 修正後 |
|---|---|---|
| 沖縄 | row 11, col 0 | row 12, col 0（offsetX なし） |

**効果試算**（計算済み）:
- 鹿児島(h:2) 下端 y=550 → 沖縄(row12) 上端 y=552 → **2px 空き（衝突解消）**
- 視覚的に狭い場合は row 13（y=598）への変更を検討する

> **注意**: 四国が row 12 に移動するため、高知(row12,col3)・徳島(row12,col4) と沖縄(row12,col0) は
> 同 row に共存するが X は離れているため問題なし
